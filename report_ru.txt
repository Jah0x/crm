Обнаружены проблемы при установке зависимостей и запуске проекта:

1. **`npm install` — 403 Forbidden**
   - npm не может скачать пакеты из `registry.npmjs.org`. Чаще всего это связано с блокировкой внешнего трафика или настройками прокси.
   - Проверьте настройки сети и наличие переменных окружения `HTTP_PROXY` / `HTTPS_PROXY`.
   - Попробуйте использовать зеркала реестра (например, `https://registry.npmmirror.com`) или настроить локальный кеш `npm`.

2. **`npm run dev` — `vite: not found`**
   - Команда `vite` отсутствует, потому что зависимости не установлены.
   - После успешного `npm install` запуск `npm run dev` должен поднять dev‑сервер.

3. **`npx tsx api.test.ts` — 403 Forbidden**
   - Аналогично первой ошибке: npx пытается скачать пакет `tsx` из npm и получает 403.
   - Установите зависимости (`npm install`) или загрузите `tsx` локально и запустите тесты напрямую.

До устранения ограничений сети lint и тесты не выполняются.
